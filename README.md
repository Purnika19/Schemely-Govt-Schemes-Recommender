# Schemely-Govt-Schemes-Recommender
Schemely is a AI based government scheme recommender that matches users with the correct central and state welfare programs using a simple profile. It filters policies by age, gender, education, state and tags like Women or Startup. A scoring model ranks the top schemes, and users can download results as a PDF. Built with Flask and pandas.

Features
1. 100% automatic scheme matching
2. Age, gender, education, and state-based filtering
3.Smart scoring system for ranking
4. Tag-based relevance boost (Student, Farmer, Women, Startup etc.)
5. Direct application links
6. PDF export download
7. Works with verified scheme datasets
8. Lightweight Flask backend

*Tech stack*

React / HTML / CSS – Frontend UI

Node.js + Express – Authentication backend (Login/Signup)

Flask (Python) – Recommender & API services

Pandas – Dataset cleaning & filtering

ReportLab – PDF generation
